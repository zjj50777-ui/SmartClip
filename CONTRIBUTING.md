# Contributing to SmartClip

Thanks for your interest in contributing!

## Getting Started

1. Fork the repo and clone it locally
2. Create a virtualenv: `python -m venv venv && venv\Scripts\activate` (Windows)
3. Install dev deps: `pip install -r requirements.txt ruff`
4. Make your changes on a feature branch
5. Run `ruff check smartclip/` to lint
6. Submit a PR to `main`

## Code Style

- Follow PEP 8
- Use type hints where practical
- Keep functions small and focused
- Add docstrings for public APIs

See [README.md](README.md#roadmap) for planned features. PRs aligned with the roadmap get priority review.
